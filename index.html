About JSCalc

×
JSCalc is a Web calculator built using only HTML, CSS and Javascript.
The calculator has shortcuts to make it easier for the user to enter input:
  • numbers from 0 to 9 to type a number
  • * to multiply
  • - to subtract
  • + to add
  • c / C or ESC to delete the display content
  • / to divide
  • ENTER to get the result of the operation
